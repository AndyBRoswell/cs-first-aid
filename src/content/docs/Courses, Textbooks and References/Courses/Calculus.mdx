# 院校开课情况选讲

一般而言，《微积分》开设 2 个学期（semester）/学段（period），主要内容分别是单变量微积分与多变量微积分。部分院校可能会拆成 3 个学期/学段，矢量微积分等内容常安排在第 3 学期/学段。
