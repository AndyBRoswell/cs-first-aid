# cs-first-aid

最后更新：2021/4/10

这是为在中国内地就读计算机类专业的本科生准备的一份自救指南，顶校学神以及其他任何天赋异禀人士请自觉跳过。

计算机类专业包括但不限于：计算机科学与技术、软件工程、信息安全、网络安全、智能科学与技术、人工智能，等等。

# 前言&emsp;在那内卷时代，没有置身于事外

本部分最后更新于：2021/2/11

相信许多人都有着高考失利未能进入理想大学的遗憾，也深知自己在大学里能够获得的资源远远少于顶尖院校的各路学霸。

在这个越来越卷的时代，无数同学如履薄冰、步步为营，生怕在某个将自己导向不同世界线的关口做出错误的选择，在发现走错了的时候却已追悔莫及。

容错率越来越低：学历歧视、35 岁危机、加班严重、全年无休……

作为在大内卷时代摸爬滚打的一员，私以为有必要将自己踩过的坑以及总结出来的经验与方法提供给后来者，帮助各位尽可能利用好最珍贵的本科时光，为日后的自己积累更多的选择筹码。所谓 “一步错，步步错” ，希望这份《自救指南》也能给到大家一些及时止损的机会，降低在做错了选择以后继续错下去的概率。

本《自救指南》将长期更新。也欢迎大家批评指正。

注意：

撰写本《自救指南》的主要目的，是为了让大家真正学到有用的知识与技能，考研等各类考试并不是《自救指南》主要针对的场景。

# 第零章&emsp;提升效率的工具与思想

本部分最后更新于：2021/4/10（未完成）

## 计算机

### 多花一点钱换来随后若干年的时间效率提升，是一本万利的。
如果你不想：
- 稍微开多点网页就卡，或者爆内存，使得很多网页或程序不能同时开着，学习工作起来巨麻烦。
- 卡顿到作业都不好做。

如果想给老机器升级，首选部件是内存或 SSD。特别是 SSD，它对用户体验的提升是非常明显的。

不玩单机游戏，三四千来块的机器通常也够做开发了，但是内存和硬盘的容量一定要管够。

尽量不要选带低压 U 的笔记本。

如果经济实在困难，可以考虑常去学校机房。

这里特别要说一下，有的同学家里明明是有经济能力的，但是不知道为什么，家长就是不肯让子女用稍微贵一点的机子。我看到过一些读机械、建筑之类的设计类专业的学生的家长，又不是没钱，却非得卡着三千多的预算。这样的后果就是作业都难以做完。总是不交作业（在老师检查或有学生互评的情况下）或不按要求完成作业会是什么结局，不用我多说了吧。

为了进一步提升生产力，可以考虑：
- 选用大屏的笔记本，不要怕重。有条件的同学使用台式机亦可。但要注意配置 UPS，防止宿舍停电导致丢失数据。
- 多个屏幕。可以购买外接显示器，或者使用平板电脑作为第二屏。推荐 Android 系统上的 Splashtop Wired XDisplay，免费的。
- 经济能力足够的同学可以考虑使用两台计算机，一台高性能，可以是又厚又重的游戏本（砖头本）或台式机，一般放在宿舍；另一台主打轻薄，带去课室或者其它地方，方便上课做笔记或干活。

让学习工作生活更方便的其它计算机配件：
- USB 分线器（集线器），即 USB Hub。扩展USB接口数量，使得可以同时连接 U 盘、移动硬盘、手机、平板电脑、读卡器等设备。建议配合 USB 延长线一同使用。
- 置物架。充分利用 z 轴空间。
- 排插。一定要买插座数量多的，省得插各类充电器之类的玩意儿的时候拔来拔去。
- 数位板。打草稿与画作业的各种乱七八糟的图用。我们不是专业画画的，也没必要买太贵的板。

## 电子书

- 方便做笔记，尤其是修改笔记。
- 易于备份。
- 节省宿舍空间。

有条件的情况下，建议对电子书做 OCR（光学字符识别）。各大软件的 OCR 功能一般都是收费的。

有的教材难以买到电子版，可以先买纸质版，然后在某宝上找店家扫描成电子版并识别文字。

在 PDF 上做笔记，推荐 Drawboard PDF。有免费和收费的版本，但免费版本已经够用。

选用这个软件的一个重要原因是：其它很多软件有一个严重问题：如果增加了笔记又删除，保存的 PDF 文件的大小不会减小。这个软件虽然默认情况下也是这样的，但可以通过在保存 / 另存为时调节选项来压缩其大小。

ABBYY FineReader PDF 在保存时会自动压缩到尽量小，但该软件在未输入序列号时只能保存 100 页 (?)。

## 一些重要思想

### 用钱买时间
对于能够提升学习效率的物件，确实有需要就不要犹豫，尽快买。不要老是等什么双十一双十二之类的活动，而且现在很多东西，特别是数码产品，在这些活动之前经常会悄咪咪涨价，活动一开始再降下来，制造大力度打折的假象。

不要去抢那些只有零头的优惠。比如学校附近某个奶茶店开张，为了几块十来块的优惠去排队一个下午。真的很没有必要。

我做志愿活动期间，看到过有商家在某小区门口搞活动，奖品无非是粽子或者其它一些吃的，或者一二十块这种程度的返现，结果那小区里外好多人（特别是老人家）就在那里折腾了一上午。我都替他们觉得累。

### 千万不要高估自己的记忆力
能记下来的就记下来，特别是那些主要靠记忆的、零散的知识点。因为有可能因为失眠或者其它各种古怪的身体原因而使得记忆受影响，或者发现自己其实不是很适合这一行，相关的知识不是很好记。

还有一个重要理由，就是许多书的信息密度其实比较低。自己整理笔记，提高信息密度，日后想复习想查阅都很高效。

### 记笔记的效率
对于一些信息密度较低的书，可以开一个新的文档去记笔记。如果书的信息密度比较高，那么建议直接在书上画重点。

有条件的同学可以使用语音输入。

### 不要光学不练
尤其是学习编程语言期间，必须通过做项目来巩固知识，否则很快会遗忘。

想不到要做什么项目的，可以每天做一些数学题，微积分之类的；也可以提前去刷 LeetCode，或者到各大学校的 Online Judge（OJ）上做算法题。

### 自动化
觉得用计算机做某件事很麻烦，就可以去想怎样将其自动化。常见的途径是：搜索相关的软件，或自己编程实现。

### 熟悉生产力工具的使用
比如各类IDE（集成开发环境）、Microsoft Office、LaTeX，等等。这部分时间是不能省的。不愿意额外花时间去学这些东西，短期来看虽然节省了时间，但从长期来看，带来的麻烦在时间上的开销大得多。

同样地，如果实在不想跟相关的教程去学，那么可以试着：每当用这些软件做某项工作觉得很麻烦时，就去网上搜索可能提高效率的操作方法。

# 第一章&emsp;课程、教材与公开课

本部分最后更新于：2021/4/10（未完成）

内地与境外高校计算机类专业的总体差距，主要体现在打基础的本科教学上（这个观点是我们老师说的，他也是教研室主任）。而内地的计算机顶校与非顶校之间，本科教学质量的差距也是十分值得引起重视的。

通过充分利用信息时代以来最伟大的发明——互联网，大家也有机会享受（至少是一部分）以往只有顶校学生才可以享受到的资源。

在没有特殊说明的情况下，对于每门课，只需要从罗列的若干本教材中选择一本即可（不包括习题集）。

对于公开课，没有注明建议倍速，并不代表应当原速播放。可能是为了照顾基础不好的学生，许多大学在录制公开课时，都会将视频的播放速度降低。大家在播放的时候，一般都可以加速播放。

如无意外，应当在不晚于 “建议开课学期” 完成相应课程的学习。

# 第二章&emsp;我应该参加哪些活动？

## 竞赛

以 ACM-ICPC 及相似的比赛为例。在中学有算法（OI）基础的同学可以试着去继续参加。如果之前完全没有基础，大学才起步，风险就很大了。

信安那边的 CTF 没了解过，不作建议。

## 科研
想走学术保研道路的，本科可以考虑尽早进组。但是许多学校的学术保研名额都很有限。

## 开源项目
不想做科研而想去工业界的，应当尽早开始在 GitHub 等开源网站上做自己的项目。

但按照各高校和科研院所的卷况，除非天赋上特别优异，否则我不建议走科研这条路。绝大多数人更适合去搞工程。按现在的大环境，如果更多人能去工业界，既有利于减小学术界的内卷程度，减少灌水论文与专利等 “成果” 的产生，还能促进超前的科研成果尽快落地。

## 大创、开放性实验等
假如想做大创（大学生创新创业训练计划）项目，建议先从学长处了解，自己学校报销经费的流程是否繁琐。如果过于繁琐，则大创项目不一定要作为首选。

需要留意，学校可能会要求每一个本科生修得一定数量的 “创新学分”，否则不准予毕业。这类创新学分往往需要通过参加大创项目和开放性实验获得。如果有这样的要求，那么就要尽早完成这些项目，把该拿的学分拿好。

## 志愿者
参加志愿活动，积累一定数量的志愿时。志愿活动可能是一些高校（特别是境外高校）招录时的硬性指标。

# 第三章&emsp;哪些事情应当尽早完成、尽早留意或尽早开始？

## 日后的发展方向
是本科毕业完就直接工作，还是说要继续深造；是考研保研，抑或是申请境外的学校，都应当尽快抓定主意，提前做好准备。

## 英语
越早开始抓越好。

英语的语法在高中基本都已经学完了，后续主要是需要不断积累词汇量。从大一甚至大一入学之前就可以开始每天背单词了。开学以后，尽快考完四六级。有留学意向的同学也要尽可能早地开始准备 IELTS 或 TOEFL 等考试。

除了升学要求，如果英语水平不过关，在计算机行业也会很吃亏。从进入这一行（包括学生阶段）开始，就应当让自己多接触英语环境。具体方法主要有：
- 将（PC 或手机、平板的）操作系统和常用软件的语言改为英文。
- 多看英文网站、英文资料。例如，使用 Google 搜索，并且将搜索的默认语言改为英文。做项目的过程中，多看英文文档和其它英文资料。

## GPA
对于非计算机类专业或大类的同学，有转到计算机类专业的意愿的，就需要保持自己的 GPA 在前列，因为很多学校对转专业的要求就包括 GPA 排名靠前。

如果拿不定主意是否要去留学，就应当按照确定需要留学的方向去走。对 GPA 有高要求的院校不计其数。万一之后决定去留学，但又发现 GPA 可能不够，就晚了。虽然有可能可以通过重修来刷新成绩，但不是所有学校都允许这样做。而且重修是要额外收费的。
